About Commander.io

Commander.io is a universal remote control for robots and the Internet of Things. It is a mobile app that can be used to send commands to your robots or other connected devices.

It can communicate with any device that supports the Common Protocol for Programming Physical I/O.

Cylon.js, Gobot and Artoo are frameworks that implement an API compatible with this protocol.

Commander is open source software developed using PhoneGap and Ionic Framework.

You can check out the source code on Github. We look forward to your contributions!

Command Sets

A command set is a preconfigured user interface that knows about the commands for a particular kind of device. There are several different types of layouts, depending on your needs.

List

A simple list of commands, that can make things happen in the real-world

D-Pad

The "D-pad" or "Directional Pad" is a great way to control things that move in 2 dimensions

Joystick

The joystick is the most common user interface for things that move in 3-dimensions. Very useful for drones!
