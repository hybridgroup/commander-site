Commander

A mobile app to control robots and connected devices.

About Commander

Commander is a mobile app to control robots, connected devices, and the Internet of Things.

Commander can be used to send commands to your robots or listen for events coming from them.

You can load a command set that is a list of the preconfigured commands and you can choose between different types of layouts.

It can communicate with any device that supports the Common Protocol for Programming Physical I/O.

Cylon.js, Gobot and Artoo are frameworks that implement an API compatible with this protocol.

Commander is open source software developed using PhoneGap and Ionic Framework. You can check out the source code.

Command Set Types

List

Start with a simple list of commands to make things happen in the real-world

D-Pad

The "D-pad" or "Directional Pad" is a great way to control things that move in 2 dimensions

Joystick

The joystick is the most common user interface for things that move in 3-dimensions. Very useful for drones!
